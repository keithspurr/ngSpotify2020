About ngSpotify

Version: 2.1

ngSpotify Version 2.1 has been updated to Angular 9.

The Spotify Web API went private in 2016.

ngSpotify pulls the Private Spotify Web API and automatically refreshes user tokens.

Search for bands by name, or for musicians by name.
Click your result of choice from the query list to display albums and track listings.

Level three also gives track previews.
Enjoy!
